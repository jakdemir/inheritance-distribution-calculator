# Inheritance Distribution Calculator

A tool for calculating inheritance distribution based on family tree structures.

## Overview

This calculator helps determine the distribution of inheritance among family members according to specified rules and family relationships.

## Features

- Family tree visualization
- Automatic inheritance calculation
- Support for various inheritance scenarios

## Technologies

- Python (Backend)
- React/TypeScript (Frontend)

## Project Structure

```
backend/    # Python API server
frontend/   # React web application
```

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```
